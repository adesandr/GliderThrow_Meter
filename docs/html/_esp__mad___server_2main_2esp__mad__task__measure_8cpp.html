Detailed Description

This task process the measure and calibration of the MPU6050.

Author

This task handles the periodicall MPU6050 measurement and computes the angle and travel of the wing surface. This file is part of the esp_mad project. travel, chordSurface and angle are globals and used by the task_server (see esp_mad_task_http_server.c) to respond to the periodical ajax script 'get' request done by the esp.html page.
