\doxysubsection{Detailed Description}
This file include all the code to deal with a simple http server.

H\+T\+TP server deal with the various http request from the clients. At the fist reception of an http get from one client, the server respond with the requested uri (see Website\+Files/ for the various uri). esp\+\_\+mad\+\_\+task\+\_\+http\+\_\+server deals with \+:
\begin{DoxyItemize}
\item nvs intialisation
\item dhcp server intialisation
\item wifi driver initialisation in soft AP mode
\item when AP station is started, the http server is launched and the uri handles are setup Uris are embedded in the .rodata D\+R\+AM segment (see C\+Make\+List.\+txt) Main H\+T\+ML page is Website\+Files/esp.\+html and used bootstrap framework and jquery. Ressources for bootstrap and jquery are minified version in Website\+Files/
\end{DoxyItemize}

Version history 01/07/2019 \+: maxi, mini, travel for both sensors added 24/03/2020 \+: http\+\_\+uri\+\_\+t variables declaration post-\/fixed by \char`\"{}\+\_\+uri\char`\"{} to insure compatibility with E\+S\+P-\/\+I\+DF 4.\+0 and C\+Make build System buffer buf\mbox{[}\mbox{]} in chord\+\_\+post\+\_\+handler pass to 50 bytes.
